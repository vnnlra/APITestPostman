# Registrazione a Postman

Per utilizzare **Postman** è necessario effettuare la registrazione con un account.

## Procedura di registrazione
1. Avviare Postman.
2. Nella schermata iniziale scegliere una delle opzioni di accesso disponibili:
   - **Google**
   - **GitHub**
   - **SSO** (se fornito dall’istituto o dall’organizzazione)
3. Completare l’accesso seguendo le istruzioni del servizio scelto.

Non è necessario creare un account Postman separato se si utilizza Google o GitHub.

## Dopo il login
Una volta effettuato l’accesso:
- è possibile creare e salvare richieste HTTP
- le richieste vengono associate al proprio account
- Postman è pronto per essere usato come client per il testing delle API

Per le attività di laboratorio verranno utilizzate solo le funzionalità di base.
